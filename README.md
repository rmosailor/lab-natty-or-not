# Apresentação sobre a evolução do delphi

## Descrição

Esta apresentação com 10 slides apresenta uma visão a respeito das inovações que a ferramenta DELPHI apresentou na sua história desde a criação

## Tecnologias Utilizadas 

- IA Generativa **[Gemini] (https://gemini.google.com) para gerar o prompt 
- IA Generativa **[Gamma.app] (https://gamma.app) para gerar a apresentação baseada no prompt gerado pelo Gemini

## Processo de geração

Usei o "Gemini" par agerar o prompt;
Copiei o prompt gerado e usei como imput para o "Gamma" gerar a apresentação.

## Resultados 

Foi criado uma apresentação estilizada para os 30 anos de história da ferramenta
